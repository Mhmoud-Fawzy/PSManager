# PlayStation Manager - مدير محل PS

تطبيق ويب لإدارة محل ألعاب البلايستيشن، يوفر نظام شامل لتتبع جلسات اللعب وإدارة المستخدمين.

## المميزات ✨

- 🎮 **إدارة جلسات الألعاب** - تتبع جلسات اللعب والوقت المستهلك
- ⏱️ **عداد الوقت** - قياس دقيق لمدة الجلسة
- 💰 **حساب التكلفة** - حساب مبلغ الدفع تلقائياً بناءً على الوقت
- 📱 **واجهة سريعة الاستجابة** - تصميم متوافق مع جميع الأجهزة
- 🌙 **واجهة داكنة احترافية** - تصميم حديث وسهل الاستخدام

## التقنيات المستخدمة 🛠️

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- JavaScript Vanilla
- Font Awesome Icons
- Google Fonts (Cairo)

## البدء السريع 🚀

1. انسخ المستودع:
```bash
git clone https://github.com/Mhmoud-Fawzy/PSManager.git
```

2. افتح الملف في المتصفح:
```bash
index.html
```

## الاستخدام 📖

- أضف جلسة لعب جديدة
- اختر المتسابق والوقت
- احسب التكلفة تلقائياً
- عرض السجل الكامل للجلسات
